# ------------------------------------------------------------------------------
# 1. SDK OFICIAL DE GOOGLE GEMINI
# Descripcion: Es la libreria oficial y mas moderna de Google para conectar 
# aplicaciones de Python con los modelos de Inteligencia Artificial.
# Utilidad en el codigo: Permite crear el cliente (genai.Client), enviar los 
# prompts de texto, subir la foto del horario escolar a la nube de Google 
# (File API) y mantener la memoria de la conversacion.
# ------------------------------------------------------------------------------
google-genai

# ------------------------------------------------------------------------------
# 2. GESTOR DE VARIABLES DE ENTORNO
# Descripcion: Herramienta de seguridad para ocultar datos sensibles.
# Utilidad en el codigo: Busca y lee el archivo oculto ".env" para extraer tu 
# GOOGLE_API_KEY. Esto evita que tu contrasena quede expuesta directamente 
# dentro del codigo fuente (hermes_app.py).
# ------------------------------------------------------------------------------
python-dotenv

# ------------------------------------------------------------------------------
# 3. MAPEO OBJETO-RELACIONAL (ORM) PARA BASE DE DATOS
# Descripcion: Es una libreria avanzada para gestionar bases de datos SQL 
# usando programacion orientada a objetos en lugar de consultas manuales.
# Utilidad en el codigo: Se encarga de crear el archivo "hermes_unificado.sqlite", 
# construir las tablas de usuarios y clases, y guardar la informacion que 
# la IA extrae de tu horario de forma segura y estructurada.
# ------------------------------------------------------------------------------
SQLAlchemy

# ------------------------------------------------------------------------------
# 4. CAPTURA DE AUDIO (MICROFONO)
# Descripcion: Modulo para grabar y procesar audio de hardware en tiempo real.
# Utilidad en el codigo: Enciende tu microfono cuando presionas el boton "Voz", 
# graba lo que dices durante 5 segundos y lo mantiene en la memoria de la PC.
# ------------------------------------------------------------------------------
sounddevice

# ------------------------------------------------------------------------------
# 5. PROCESAMIENTO MATEMATICO DE MATRICES
# Descripcion: Libreria fundamental para calculos cientificos y arrays.
# Utilidad en el codigo: El audio que captura "sounddevice" no es sonido puro, 
# son miles de numeros (frecuencias). Numpy procesa esta inmensa matriz de 
# numeros casi al instante para que no se congele tu computadora.
# ------------------------------------------------------------------------------
numpy

# ------------------------------------------------------------------------------
# 6. CONVERSION DE AUDIO CIENTIFICO
# Descripcion: Modulo del ecosistema SciPy para interactuar con archivos.
# Utilidad en el codigo: Toma la matriz de numeros que genero Numpy y la 
# "empaqueta" para construir y guardar un archivo de audio real (.wav) 
# que posteriormente se envia a la Inteligencia Artificial.
# ------------------------------------------------------------------------------
scipy

# ------------------------------------------------------------------------------
# 7. SINTESIS DE VOZ DE GOOGLE (TEXT-TO-SPEECH)
# Descripcion: Herramienta que traduce texto escrito a lenguaje hablado.
# Utilidad en el codigo: Cuando la IA responde con un mensaje de texto, gTTS 
# toma esas palabras y genera un archivo de audio (.mp3) con la voz del 
# asistente (Hermes) para que puedas escucharlo.
# ------------------------------------------------------------------------------
gTTS

# ------------------------------------------------------------------------------
# 8. REPRODUCTOR MULTIMEDIA ASINCRONO
# Descripcion: Modulo de audio extraido de una libreria de creacion de juegos.
# Utilidad en el codigo: Se encarga de reproducir en las bocinas el archivo 
# .mp3 que genero gTTS. Se utiliza esta libreria porque permite reproducir 
# el sonido "de fondo" sin bloquear ni congelar la interfaz grafica.
# ------------------------------------------------------------------------------
pygame

uuid
